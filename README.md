# Line Follower Robot

## Project Description

This repository contains documentation for a team robotics project involving the design, assembly, and testing of a line-following robot.

The robot follows a predefined path using infrared sensors and an ATmega64A microcontroller. The project provided practical experience in robotics, electronics, assembly, testing, and troubleshooting.

## Robot Operation

The robot uses infrared sensors to detect a black path on a light-colored surface.

* Move forward when the center sensor detects the line.
* Turn left when the left sensors detect the line.
* Turn right when the right sensors detect the line.
* Stop or search for the path when no sensor detects the line.

Motor speed is controlled using PWM (Pulse Width Modulation).
